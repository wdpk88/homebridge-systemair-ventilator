# Homebridge Systemair Ventilator Plugin

This is a Homebridge plugin for controlling a Systemair Ventilator through its web interface. You have to have connected the SAVEConnect WIFI for this to work. You just need the wifi adresse for the SAVEConnect wifi, and add this in the plugin settings.

## Installation

1. Install Homebridge:
   ```bash
   npm install -g homebridge
